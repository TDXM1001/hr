二、路由结构
/login                           # 登录页（公开）

│
├── /employee                    # 员工管理
│   ├── /employee/list           # 员工列表（分页表格 + 高级搜索）
│   ├── /employee/detail/:id     # 员工详情（Tab 页：基础/合同/证件/联系人/变更记录）
│   ├── /employee/add            # 新增员工（分步表单）

├── /attendance                  # 考勤管理

前端设计要点总结：

✅ 经典后台管理布局（侧边菜单 + 标签页 + 面包屑）
✅ 动态路由 + 按钮级权限控制（v-permission 指令）
✅ 组件化设计：通用表格、表单、树选择器等
✅ 组合式函数（Composables）提取可复用逻辑
✅ 字典缓存机制，减少重复请求
✅ 组织架构图可视化 + 排班日历视图
